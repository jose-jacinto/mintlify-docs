## What is Opsmatic Analytics?

Opsmatic Analytics provides comprehensive monitoring and insights for your automation workflows across n8n and Make.com platforms. Get real-time performance metrics, historical trends, and actionable insights to optimize your automation efficiency.

## Data Collection and Processing

### How Data is Collected

<Steps>
  <Step title="API Monitoring">
    Opsmatic regularly polls your connected platforms for workflow and execution data
  </Step>
  <Step title="Real-time Sync">
    Data is synchronized every 5-15 minutes depending on your plan and platform activity
  </Step>
  <Step title="Processing Pipeline">
    Raw data is processed, aggregated, and stored for fast dashboard performance
  </Step>
  <Step title="Analytics Generation">
    Metrics are calculated and trends are identified for dashboard display
  </Step>
</Steps>

### Data Retention Policy

<CardGroup cols={3}>
  <Card title="Raw Data" icon="database">
    **Execution Logs**: 30 days detailed
    **Availability Metrics**: 7 days raw data
  </Card>
  <Card title="Aggregated Data" icon="chart-bar">
    **Daily Summaries**: 1 year retention
    **Weekly/Monthly**: 2 years retention
  </Card>
  <Card title="Historical Trends" icon="trend-up">
    **Long-term Patterns**: Indefinite retention
    **Performance Baselines**: Permanent storage
  </Card>
</CardGroup>

## Setting Up Analytics Monitoring

### Automatic Setup

Analytics monitoring begins automatically when you:

1. **Connect a Platform**: Add your first n8n or Make.com connection
2. **Verify Connection**: Test and confirm the connection works
3. **Wait for Data**: Allow 15-30 minutes for initial data collection
4. **Access Analytics**: Navigate to the Analytics dashboard

## Troubleshooting Analytics

### Common Issues

<AccordionGroup>
  <Accordion title="Missing Data">
    **Possible Causes:**
    - New connection (data collection starting)
    - Connection authentication issues
    - Platform API limitations
    - Network connectivity problems
    
    **Solutions:**
    - Wait 15-30 minutes for new connections
    - Test and fix connection issues
    - Check platform API status
    - Verify network accessibility
  </Accordion>
  
  <Accordion title="Incorrect Metrics">
    **Possible Causes:**
    - Timezone mismatches
    - Data sync delays
    - Platform clock differences
    - Caching issues
    
    **Solutions:**
    - Verify timezone settings
    - Allow time for data synchronization
    - Refresh dashboard manually
    - Clear browser cache
  </Accordion>
</AccordionGroup>
